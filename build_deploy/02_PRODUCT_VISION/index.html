02 · PRODUCT VISION

Версия: 5.0 | Дата: 05.03.2026 | Статус: Public

Полное видение платформы Wellex: Emotional Mining, двойная архитектура APP/WEB, Multi-EVM, Card On-Ramp, WHOOP-модель комьюнити, 3-летняя стратегия

Миссия​

Wellex создаёт индустрию Emotional Mining — процесс, где забота о здоровье и эмоциональном благополучии конвертируется в финансовый актив. Мы строим первую в мире экосистему, объединяющую биометрический браслет, AI-коучинг, сообщество единомышленников и DeFi-yield в единый протокол.

Emotional Mining — новая индустрия​

Определение​

Emotional Mining — процесс генерации экономической ценности из верифицированных данных о физическом, эмоциональном и ментальном благополучии человека.

Почему это индустрия, а не фича​

КритерийEmotional Mining
Уникальная ценностьВпервые эмоциональное благополучие имеет прямую финансовую стоимость
Технологический стекBiosensors + ML emotion detection + DeFi + Blockchain oracle
МасштабируемостьЛюбой человек с wearable может «майнить» — 1B+ потенциальных участников
Экономическая модельVerifiable wellness data → on-chain proof → yield distribution
АналогияBitcoin Mining: электричество → вычисления → ценность. Emotional Mining: забота о себе → биометрия → ценность

WVI как «Hash Rate» Emotional Mining​

В Bitcoin Mining hash rate определяет доход. В Emotional Mining — WVI (0–100) определяет Monthly Yield:

  • WVI — это ваш «hash rate»
  • Браслет — это ваш «mining rig»
  • Забота о здоровье — это ваша «электроэнергия»
  • Monthly Yield — это ваш «блок-реворд»

Двойная архитектура: APP vs WEB​

Фундаментальный принцип разделения​

APP = Wellness + Community + Мотивация. Абсолютно чистый wellness-продукт. WEB = Finance + Vault + MLM + Full Yield Management. Полный финансовый функционал.

Это не просто два интерфейса — это две различные философии продукта, объединённые единым аккаунтом.

Grace Days — гибкость без наказания​

Проблема: WVI = 0 при пропуске дня — жёсткая механика, создающая стресс и отток.

Решение — 3 Grace Days в месяц:

  • В Grace Day: WVI = среднее за последние 7 активных дней
  • Пользователь видит: "📅 Grace Day использован (2 из 3 осталось)"
  • При болезни/путешествии: AI Coach автоматически предлагает активировать Grace Day
  • Grace Days не накапливаются (сбрасываются 1-го числа)

WVI Floor для стейкеров:

  • Минимальный WVI при активном стейкинге = 25 (не 0)
  • При полном отсутствии ношения → WVI 25 = 0% yield (не штраф — просто нет бонуса)
  • Психология: "ты не потерял — ты просто не заработал сегодня"

Offline Buffer — 48 часов автономии​

Браслет хранит до 48 часов биометрических данных без связи с телефоном. При восстановлении соединения — автоматическая синхронизация. Пользователь не теряет WVI при отсутствии связи.

Применение: авиаперелёты, зоны без покрытия, разряженный телефон.

AI Yield Engine — встроен в кошелёк​

Каждый ERC-4337 Smart Wallet содержит AI Yield Engine — автономный оптимизатор доходности:

  • 800+ стратегий из 8 источников (DeFi Lending + RWA + Liquidity Mining + Options Strategies + Basis Trading + Delta-Neutral + Cross-Chain Arbitrage + Emerging Protocols)
  • 1M+ параметрических комбинаций оптимизировано ML-методами (gradient boosting + backtesting)
  • Пересчёт каждые 15 минут, latency принятия решения <500мс
  • WVI пользователя определяет аллокацию: низкий WVI = консервативный портфель, высокий = агрессивный

Пользователь не управляет стратегиями вручную. Всё автоматически. → 07_AI_SYSTEM.md, 19_WHITEPAPER_v3.md секция 9.3

⚠️ Target Monthly Yield 0–20% — не гарантия. Среднее по backtest (36 мес): 0.6–11.4%/мес в зависимости от WVI-тира. Фактический yield зависит от рыночных условий.

Мировой рынок wellness оценивается в $6.3 трлн (Global Wellness Institute, 2025) и растёт на 10% ежегодно. Рынок DeFi преодолел $195B+ TVL. Рынок носимых устройств достиг $82B. Но все три индустрии существуют изолированно, и пользователь теряет на каждом стыке:

1.2 Решение​

Wellex — первый протокол, объединяющий биометрику, DeFi и AI в одну экосистему, где здоровый образ жизни напрямую конвертируется в финансовый доход.

Смарт-браслет (бесплатно при подписке $19/мес) → Биометрия → Wellex Vitality Index (0–100) → Доходность по депозиту 0–20% в месяц

Ключевые отличия от M2E:

  • Доход генерируется 8 источниками низко/средне рискового крипторынка: DeFi Lending (Aave, Compound, Morpho), RWA (Ondo, Maple — US Treasuries), Prediction Markets LP (Polymarket, Azuro) + опционные стратегии, Арбитраж (funding rate BTC/ETH, CEX/DEX spread)
  • Wellex App (iOS/Android) не является криптокошельком — чистый wellness. Все финансовые операции — исключительно в WEB-версии.
  • Wellex Vitality Index усиливает доходность, но не является её источником
  • AI Wallet Agent автоматически оптимизирует стратегию
  • Модель не зависит от притока новых пользователей
  • Браслет бесплатный — порог входа минимален

3. WELLEX VITALITY INDEX (WVI)​

3.1 Что такое WVI​

Wellex Vitality Index (WVI) — единая числовая оценка здоровья пользователя от 0 до 100, рассчитываемая ежедневно на основе данных смарт-браслета. WVI напрямую определяет ежемесячная доходность.

3.2 Три компонента WVI​

WVI (0–100) формируется из трёх компонентов: 🧠 Эмоции (35%), 😴 Recovery (35%), 🏃 Нагрузка (30%).

Компонент 1: Эмоциональное состояние (35%)​

МетрикаДиапазонОптимумСенсор
HRV (RMSSD)10–150 мс>60 мсPPG
Стресс-индекс0–100<30PPG (LF/HF ratio)
Эмоциональный баланс0–100>70HRV + circadian rhythm

Формула: E_score = 0.45 × norm(HRV) + 0.35 × (100 − stress) + 0.20 × balance

Компонент 2: Восстановление (35%)​

МетрикаДиапазонОптимумСенсор
Длительность сна0–12 ч7–9 чАкселерометр
Глубокий сон0–100%>20%PPG + акселерометр
Recovery score (утро)0–100>70Утренний HRV
Температура (δ)−2..+2°C±0.3°CТермометр

Формула: R_score = 0.30 × norm(duration) + 0.30 × deep_pct + 0.25 × recovery + 0.15 × temp_stability

Компонент 3: Физическая нагрузка (30%)​

МетрикаДиапазонОптимумСенсор
Шаги/день0–30,0008,000–12,000Акселерометр
Активные калории0–2,000300–600Акселерометр + PPG
Тренировки (мин)0–18030–60Гироскоп + PPG
Интенсивность (avg HR%)0–100%60–80% max HRPPG

Формула: P_score = 0.25 × norm(steps) + 0.25 × norm(calories) + 0.30 × norm(training) + 0.20 × intensity

3.3 Итоговая формула WVI​

Дневной WVI = E_score × 0.35 + R_score × 0.35 + P_score × 0.30

Месячный WVI = Σ(дневной WVI за все дни месяца) / N, где N = количество дней в месяце.

Ключевой принцип: дни без ношения браслета = дневной WVI = 0. Это мотивирует носить браслет ежедневно.

3.4 Нормализация​

Все метрики нормализуются через sigmoid-подобную функцию с персонализированными порогами: norm(x) = 100 / (1 + exp(−k × (x − x_median))) Персонализация: x_median и k адаптируются к пользователю через 7 дней калибровки.

3.5 Калибровка (первые 7 дней)​

ДниЭтап
1–3Сбор baseline (нормальный режим жизни)
4–5Определение персональных порогов (HRV, сон, активность)
6–7Активация WVI с персонализированными нормами

WVI в период калибровки = фиксированный 50, yield = 5% в месяц.

9.3.2 AI-Powered Algorithmic Strategy Selection​

Задача: из 1M+ параметрических комбинаций отобрать 800+ стратегий с оптимальным Sharpe ratio при заданном уровне риска для каждого WVI-тира.

Метод: многоэтапная ML-оптимизация:

  1. Генерация кандидатов — параметрическое пространство (~1M комбинаций) генерируется по сетке параметров
  2. Фильтрация — исключение стратегий с Sharpe < 1.0, Max Drawdown > 15%, ликвидностью < $5M
  3. Ранжирование — gradient boosting модель (XGBoost) оценивает каждую стратегию по 40+ признакам
  4. Ансамблирование — топ-800+ стратегий группируются по WVI-тиру и корреляционным кластерам
  5. Backtesting — каждая стратегия верифицирована на 36 месяцах исторических данных (2022–2025)

Подход аналогичен количественным методам Renaissance Technologies и Two Sigma — без квантовых вычислений, но с полной воспроизводимостью результатов.

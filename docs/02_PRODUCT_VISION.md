## Emotional Mining — новая индустрия

### Определение

**Emotional Mining** — процесс генерации экономической ценности из верифицированных данных о физическом, эмоциональном и ментальном благополучии человека.

### Почему это индустрия, а не фича

| Критерий | Emotional Mining |
|----------|-----------------|
| **Уникальная ценность** | Впервые эмоциональное благополучие имеет прямую финансовую стоимость |
| **Технологический стек** | Biosensors + ML emotion detection + DeFi + Blockchain oracle |
| **Масштабируемость** | Любой человек с wearable может «майнить» — 1B+ потенциальных участников |
| **Экономическая модель** | Verifiable wellness data → on-chain proof → yield distribution |
| **Аналогия** | Bitcoin Mining: электричество → вычисления → ценность. Emotional Mining: забота о себе → биометрия → ценность |

### Эволюция Mining

| Поколение | Модель | Ресурс | Ограничение |
|-----------|--------|--------|-------------|
| **Bitcoin Mining** | Proof of Work | Электричество + GPU | Расход энергии, централизация |
| **Emotional Mining (Wellex)** | Proof of Wellness | Эмоции + Сон + Активность | Устойчивая модель — yield из реального DeFi |

### WVI как «Hash Rate» Emotional Mining

В Bitcoin Mining hash rate определяет доход. В Emotional Mining — **WVI (0–100)** определяет **Monthly Yield**:

- WVI — это ваш «hash rate»
- Браслет — это ваш «mining rig»
- Забота о здоровье — это ваша «электроэнергия»
- **Monthly Yield** — это ваш «блок-реворд»

## Grace Days — гибкость без наказания

**Проблема:** WVI = 0 при пропуске дня — жёсткая механика, создающая стресс и отток.

**Решение — 3 Grace Days в месяц:**
- В Grace Day: WVI = среднее за последние 7 активных дней
- Пользователь видит: "📅 Grace Day использован (2 из 3 осталось)"
- При болезни/путешествии: AI Coach автоматически предлагает активировать Grace Day
- Grace Days не накапливаются (сбрасываются 1-го числа)

**WVI Floor для стейкеров:**
- Минимальный WVI при активном стейкинге = 25 (не 0)
- При полном отсутствии ношения → WVI 25 = 0% yield (не штраф — просто нет бонуса)
- Психология: *"ты не потерял — ты просто не заработал сегодня"*

## Offline Buffer — 48 часов автономии

Браслет хранит до **48 часов биометрических данных** без связи с телефоном. При восстановлении соединения — автоматическая синхронизация. Пользователь не теряет WVI при отсутствии связи.

Применение: авиаперелёты, зоны без покрытия, разряженный телефон.

# 04 · WELLNESS APP (iOS / Android)
> **Версия:** 6.1 | **Дата:** 27.03.2026 | **Статус:** Public


> Мобильное приложение Wellex: чистый wellness, комьюнити WHOOP-модели, Monthly Yield-отображение. Без стейкинга, без MLM, без финансовых операций.
>
> ⚠️ Wellex App — **не криптокошелёк.** Это wellness-приложение. Все финансовые операции выполняются исключительно в WEB-версии.
> ⚠️ Target Yield is not a guarantee. APP displays yield % only — no dollar amounts. Actual yield depends on DeFi market conditions.

---

## 1. Философия UX

### Ключевой принцип: APP = Wellness + Community

**В приложении нет (и не будет):**
- Стейкинга и любого стейкинг-интерфейса
- Депозитов, выводов, финансовых транзакций
- MLM-структуры и партнёрского дерева
- Отображения сумм в долларах
- Кошельков и балансов
- Любых финансовых операций

> **Важно:** Wellex App **не является криптокошельком.** Пользователи, использующие только браслет без стейкинга, видят статистику WVI и потенциал доходности. Для активации стейкинга и начала получения дохода — переход в WEB-версию.

**В приложении есть:**
- WVI (Wellex Vitality Index) — единая шкала здоровья 0–100
- Биометрия: эмоции, сон, активность
- AI-коуч (видео / аудио / текст)
- Community WHOOP-модели: discovery владельцев, группы, челленджи, лидерборды
- Monthly Yield-отображение (только %, без сумм)
- Кнопка перехода в WEB-версию

### Monthly Yield — правила отображения

| Статус пользователя | Отображение в APP | Действие при тапе |
|---------------------|------------------|-------------------|
| **С активным стейкингом** | Текущий Monthly Yield в процентах (напр. «15.8%»). Зелёный акцент, крупный шрифт. Без суммы в $. | Информационный блок |
| **Без стейкинга** | Дэш «—». Мелкий текст «Узнать больше →» | Popup: «Узнать больше о доходности» → deep link в WEB-версию |

### Lazy Factor

> **Открыл приложение → увидел шкалу → получил рекомендацию → закрыл. Среднее время сессии: 30 секунд.**

### Design Principles

| Принцип | Реализация |
|---------|-----------|
| **Glanceable** | WVI + Monthly Yield видны без скролла |
| **Zero-config** | Браслет подключается автоматически |
| **Proactive** | AI push-уведомления в нужный момент |
| **Community-first** | Комьюнити — полноценная секция, не дополнение (WHOOP-модель) |
| **Inclusive** | WCAG 2.1 AA; VoiceOver / TalkBack |
| **Offline-first** | Полный функционал без интернета; синхронизация при подключении |
| **No Finance** | Никаких финансовых элементов. Чистый wellness. |

## 8. User Flows

### 8.1 Onboarding (новый пользователь)

1. Скачать приложение → Splash screen
2. **Регистрация:** Email / Apple ID / Google (1 tap)
3. **Некастодиальный кошелёк:** при регистрации через email система автоматически генерирует некастодиальный кошелёк. Приватные ключи **принадлежат пользователю** и могут быть экспортированы в любое совместимое внешнее приложение. Пользователь сохраняет полный контроль над кошельком вне зависимости от платформы.
   - При регистрации через внешний Web3-кошелёк (WalletConnect / MetaMask) — собственный кошелёк пользователя.
4. Экран: «Подключите Wellex Band» → NFC tap или BLE scan → сопряжение (15 сек)
5. Краткий тур (3 экрана swipe): WVI · Community · AI-коуч
6. Начало калибровки (7 дней, WVI = 50)
7. Push: «Носите браслет 24/7 для точной калибровки»
8. Главный экран

> **Некастодиальный кошелёк:** создаётся автоматически при email-регистрации. Скрыт в APP — полная функциональность в WEB. Экспорт ключей доступен через WEB → Settings → Security → Export Private Key.

### 8.2 Ежедневный цикл (активный пользователь)

```
07:00 Push: «Доброе утро! Recovery 82. Отличный день для тренировки»
 → Действие: прочитать (5 сек)
12:00 Push: «WVI сегодня 74. Прогулка 20 мин → +5»
 → Действие: по желанию
19:00 Push: «Ваш WVI за сегодня: 76 (+3 от вчера)»
 → Действие: ничего
22:00 Push: «План сна: ложитесь до 23:00, нет экранов»
 → Действие: ничего или следовать
```

**Итого: 0–2 действия в день, 30 секунд в приложении.**

### 8.3 Переход в WEB (финансовые операции)

**Сценарий A: пользователь только с браслетом (без стейкинга)**

Пользователи, использующие только Wellex Band и не имеющие средств в стейкинге, видят:
- Статистику WVI и три компонента (эмоции, восстановление, физическая активность)
- Monthly Yield: «—» с подсказкой «Узнайте о доходности»
- Потенциал дохода при их текущем WVI: «Ваш WVI 72 → потенциал 12%/мес»

Механизм мотивации к переходу:
1. На главном экране — Monthly Yield «—», мелкий текст «Узнать больше →»
2. Тап → **Мотивационный popup:**
   ```
   ┌─────────────────────────────────┐
   │ 💰 Ваш WVI 72 = 12%/мес        │
   │                                 │
   │ При вашем уровне здоровья вы    │
   │ можете получать до 12% в месяц  │
   │ на ваши средства.               │
   │                                 │
   │ [Активировать доходность →]     │
   │ Открыть WEB-версию Wellex       │
   └─────────────────────────────────┘
   ```
3. Кнопка «Активировать доходность» → deep link в WEB
4. В WEB: авторизация (тот же аккаунт) → прямо на экран Staking
5. Полная финансовая функциональность: депозит, стейкинг, партнёрка

**Сценарий B: активный стейкер**
1. Пользователь видит Monthly Yield «15.8%» на главном экране
2. Тап → информационный блок с breakdown и переходом в WEB для управления

### 8.4 Discovery: найти владельцев браслетов рядом

1. Комьюнити → «Люди рядом с Wellex»
2. Карта / список владельцев с WVI, дистанцией, интересами
3. Тап на профиль → общие группы, WVI-график, интересы
4. [Добавить в друзья] → приглашение в команду или челлендж

### 8.5 Вступление в группу комьюнити

1. Комьюнити → Поиск → «Йога»
2. Список групп с описанием, количеством участников, avg WVI
3. Тап на группу → Preview → [Вступить]
4. Группа добавлена в «Мои группы» → доступны челленджи и feed группы

### 8.6 Участие в командном челлендже

1. Комьюнити → Активные челленджи → «Командный WVI ≥ 70»
2. Просмотр деталей: правила, сроки, участники, прогресс
3. [Присоединиться] → выбор команды или создание своей
4. Ежедневный прогресс + push-уведомления → итоговый результат

| Требование | Реализация |

## 12. Push-уведомления

| Категория | Частота | Пример | Отключаемое |
|-----------|---------|--------|------------|
| Здоровье (AI) | 2–4/день | «HRV упал — дыхание 5 мин» | Да |
| WVI прогресс | 1/день | «WVI сегодня: 76 (+3)» | Да |
| Комьюнити | 1–2/день | «Ваша команда на 2-м месте!» | Да |
| Мотивация (streaks) | 1/день | «14 дней подряд!» | Да |
| Браслет | По событию | «Батарея 10%» | Нет |
| Безопасность | По событию | «Новый вход в аккаунт» | Нет |

**Smart scheduling:** уведомления не отправляются в ночное время; часовой пояс учитывается автоматически.

## 15. Help Center & Support

### 15.1 In-App Support Flow

Пользователь обращается за помощью не выходя из приложения:

| Канал | Описание | Время ответа |
|-------|----------|-------------|
| AI Help Bot | Первая линия — отвечает на 80%+ запросов мгновенно | < 5 сек |
| In-App Chat | Живой агент поддержки (текст) | < 2 часа |
| Email | support@wellex.ai — для сложных случаев | < 24 часа |
| FAQ | Встроенная база знаний с поиском | Мгновенно |

### 15.2 Help Center — категории

▸ Браслет — подключение, синхронизация, батарея, OTA, возврат
▸ WVI — как считается, почему упал, калибровка
▸ Yield — как начисляется, когда выплачивается, почему изменился %
▸ Стейкинг — депозит, вывод, lock-up сроки (в WEB)
▸ Аккаунт — пароль, 2FA, удаление, GDPR
▸ Кошелёк — экспорт ключей, мультичейн, транзакции
▸ Партнёрская программа — ранги, бонусы, выплаты
▸ Комьюнити — группы, челленджи, модерация

### 15.3 Escalation Path

```
Пользователь → AI Help Bot (авто, мгновенно)
 ↓ если не решено (< 5% случаев)
In-App Chat → Live Agent
 ↓ если технический вопрос
Tier 2 Support (blockchain, wallet)
 ↓ если критично (потеря средств)
War Room Protocol → CTO + Head of Support
```

### 15.4 Проактивная поддержка (AI-driven)

▸ WVI упал >15 за день → автоматическая подсказка от AI Coach
▸ Браслет не синхронизировался >24 часа → push-уведомление с инструкцией
▸ Yield не начислялся →  проактивное уведомление + объяснение причины
▸ Lock-up истекает через 3 дня → напоминание с опциями (вывод / реинвест)

## 2. Регистрация / Вход в WEB

### Способ 1: Email-вход (из APP или напрямую)

1. Email + пароль → вход
2. Если пришёл из APP — аккаунт уже существует, WEB3-кошелёк создан автоматически
3. Полный доступ к финансовым функциям

### Способ 2: WEB3 Wallet

1. Кнопка «Подключить кошелёк» → WalletConnect / MetaMask / Trust Wallet
2. Подпись сообщения (EIP-4361 — Sign-In with Ethereum)
3. Ввод email для создания / привязки аккаунта Wellex
4. Полный доступ к финансовым функциям

**Единый аккаунт:** один аккаунт работает и в APP, и в WEB.

## 4. Модуль: Стейкинг (📈 Staking)

### 4.1 WellexVault — единый vault

Wellex использует **один унифицированный vault — WellexVault**. В экосистеме Wellex **отсутствуют отдельные пакеты или уровни подписки** — единый формат стейкинга для всех участников. AI-агент автоматически распределяет средства по DeFi-протоколам для максимизации yield с учётом WVI пользователя.

| Параметр | Значение |
|----------|----------|
| **Vault** | WellexVault (единый для всех) |
| **Пакеты / уровни** | **Нет** — единый формат стейкинга |
| **Управление** | AI Auto-Optimize (ручная смена стратегий недоступна) |
| **Страховое покрытие** | 100% |
| **Минимальная сумма** | $50 (Micro-Starter) |

### 4.2 Yield в зависимости от WVI

Yield пользователя определяется его **Wellex Vitality Index (WVI)**:

| WVI | Monthly Yield | Annual |
|-----|:-------------:|-------:|
| 0–39 | 0% | 0% |
| 40–49 | 2% | ~24% |
| 50–59 | 5% | ~60% |
| 60–69 | 8% | ~96% |
| 70–79 | 12% | ~144% |
| 80–89 | 16% | ~192% |
| 90–100 | 20% | ~240% |

> ⚠️ **Дисклеймер:** Фактический yield зависит от рыночных условий DeFi и распределяется пропорционально WVI из общего Yield Pool. Цифры — теоретический максимум при благоприятных условиях.

### 4.3 Lock-up по сумме депозита

| Сумма депозита | Lock-up период |
|----------------|----------------|
| $0 – $999 | 0 дней (без блокировки) |
| $1 000 – $9 999 | 30 дней |
| $10 000+ | 90 дней |

Lock-up применяется к каждому депозиту отдельно с момента его внесения.

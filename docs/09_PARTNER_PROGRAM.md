## 3. Comp Plan — 8 потоков дохода

### 3.1 Бонус за прямого реферала
5–8% от подписки каждого прямого реферала.
Мгновенная выплата, on-chain.

**Важно: база бонуса = сумма USDC после конвертации** (за вычетом комиссии on-ramp провайдера).
Пример: $1,000 через Visa → on-ramp ~3% → $970 USDC. Бонус рассчитывается от $970.

### 3.2 Командный бонус (10 уровней в глубину)

| Уровень | % от PV | Доступно с ранга |
|:-------:|:-------:|:----------------:|
| 1 | 10.0% | Explorer 🧭 |
| 2 | 7.0% | Builder 🔧 |
| 3 | 5.0% | Achiever 🏅 |
| 4 | 4.0% | Manager 📊 |
| 5 | 3.0% | Director 💎 |
| 6 | 2.0% | Leader 🛡️ |
| 7 | 1.5% | Ambassador 🌐 |
| 8 | 1.0% | Master 👁️ |
| 9 | 0.75% | Champion 👑 |
| 10 | 0.5% | Legend 🔱 |

**Динамическое сжатие:** если партнёр на промежуточном уровне неактивен, бонус «сжимается» вверх — доход вышестоящего партнёра не страдает от неактивных участников.

### 3.3 Matching Bonus
10–25% от командных бонусов прямых рефералов. Растёт с рангом: Explorer 10% → Leader 20% → Legend 25%.

### 3.4 Бонус за достижение ранга (единоразовый)

| Ранг | Бонус |
|------|:-----:|
| Explorer 🧭 | $25 |
| Builder 🔧 | $100 |
| Achiever 🏅 | $500 |
| Manager 📊 | $2,000 |
| Director 💎 | $5,000 |
| Leader 🛡️ | $15,000 |
| Ambassador 🌐 | $30,000 |
| Master 👁️ | $75,000 |
| Champion 👑 | $150,000 |
| Legend 🔱 | $500,000 |

### 3.5 WVI Performance Bonus — мост APP↔WEB

Партнёры, носящие браслет и поддерживающие высокий WVI в APP, получают множитель на все WEB-бонусы. Это **единственная точка пересечения APP и WEB** в партнёрской программе.

| WVI партнёра (из APP) | Множитель на WEB-бонусы |
|:---------------------:|:------------------------:|
| 0–40 | ×0.80 (снижение) |
| 41–60 | ×1.00 (базовый) |
| 61–80 | ×1.15 (+15%) |
| 81–100 | ×1.30 (+30%) |

### 3.6 Leadership Pool
2% от общего оборота платформы распределяется среди партнёров ранга Leader+ пропорционально объёму личной сети.

### 3.7 Fast Start Pool
1% от общего оборота — бонус для партнёров, достигших ранга Achiever в первые 30 дней.

### 3.8 Infinity Bonus (только Champion+ и Legend)
0.5% со всей сети без ограничений по глубине уровней.

### 3.9 Доля от доходности стейкинга
Партнёры, получающие доходность через WellexVault, вносят вклад в устойчивость сети:
▸ Доходность генерируется из 8 независимых DeFi-потоков (см. [06_YIELD_PROTOCOL.md](06_YIELD_PROTOCOL.md))
▸ Потоки: DeFi Lending, RWA, Liquidity Mining, Options Strategies, Basis Trading, Delta-Neutral, Cross-Chain Arbitrage, Emerging Protocols
▸ AI Yield Agent управляет 800+ стратегиями с ребалансировкой каждые 15 минут
▸ WVI Allocation Modifier: WVI 0–40 → 0–20% распределения на поток
▸ Dynamic Yield Cap: SR < 1.0 → автоматическое снижение доходности для защиты протокола
▸ Партнёрские бонусы финансируются из выручки платформы (подписки + комиссии yield-протокола), а НЕ из новых депозитов

## 4a. Архитектура смарт-контрактов для выплат

Все партнёрские выплаты исполняются через смарт-контракты на Arbitrum:

| Компонент | Описание |
|-----------|----------|
| **PayoutRouter.sol** | Маршрутизирует расчёты бонусов в выплаты USDC |

## 8. Partner WVI Lite — снятие барьера «браслет 24/7»

MLM-лидеры 40+ нередко не готовы носить браслет круглосуточно. Partner WVI Lite решает это:

▸ **Partner WVI Floor = 40** для активных партнёров (≥3 рефералов за последние 30 дней)
▸ Активный партнёр автоматически получает минимальный WVI 40 — доступ ко всем партнёрским бонусам
▸ Те, кто носит браслет и имеет реальный WVI 40+, получают множитель: ×1.15–1.30
▸ Барьер «обязательное ношение браслета» снят. Браслет = возможность, не обязательство

### Механика WVI Lite — детально

| Условие | Значение WVI | Множитель |
|---------|:------------:|:---------:|
| Нет браслета + < 3 рефералов/30д | Фактический WVI (или 0) | ×0.80 |
| Нет браслета + ≥3 рефералов/30д | **Порог 40** | ×1.00 |
| Браслет + WVI 40–60 | Фактический WVI | ×1.00 |
| Браслет + WVI 61–80 | Фактический WVI | ×1.15 |
| Браслет + WVI 81–100 | Фактический WVI | ×1.30 |

▸ WVI Lite пересчитывается ежедневно в 00:00 UTC
▸ Порог 3 реферала = рефералы с активными подписками (не просто регистрации)
▸ Партнёры видят статус WVI Lite в WEB Dashboard → Партнёрский профиль

## 1. Vision

Wellex Community — социальный слой wellness-экосистемы, доступный **только через APP**. Никаких финансовых операций, никакого MLM. Чистое wellness-сообщество с геймификацией и социальным стимулом.

> **Аналоги:** WHOOP Teams (командные метрики) + Strava Clubs (челленджи) + Duolingo Leagues (соревнования) + Peloton Community (live-события)

### Ключевые принципы

| # | Принцип | Реализация |
|---|---------|------------|
| 1 | **Wellness-first** | Все взаимодействия направлены на улучшение здоровья, не на финансовую мотивацию |
| 2 | **Privacy by design** | WVI и суб-скоры видны только с согласия. Биометрия никогда не публична |
| 3 | **Positive-only** | Нет негативных сравнений. «Ты лучше вчерашнего себя» > «Ты хуже Маши» |
| 4 | **No finance in APP** | Баланс, yield, депозиты — только в WEB. Community = чистый wellness |
| 5 | **AI-moderated** | Автоматическая модерация токсичности. AI Coach помогает с социальными целями |

## 3. Челленджи (Challenges)

### 3.1 Типы челленджей

| Тип | Описание | Частота | Участники |
|-----|----------|---------|-----------|
| **Ежедневные квесты** | 5 мини-задач (шаги, дыхание, mood check-in) | Ежедневно | Индивидуально |
| **Еженедельные тематические** | Фокус на одной области (сон, медитация, ходьба) | Еженедельно | Индивидуально / команда |
| **Командные vs Командные** | Команда vs команда по суммарному WVI | Еженедельно | Команда 5–10 |
| **Duo Challenges** | 1v1 между друзьями | По запросу | 2 человека |
| **Сезонные марафоны** | Месячные события с призовым фондом | Ежемесячно | Все |
| **Corporate Challenges** | B2B — между департаментами компании | По запросу | Корпоративные |

### 3.2 Еженедельные тематические челленджи

| Неделя | Тема | Условие | Награда |
|:------:|------|---------|---------|
| 1 | 🧘 «Неделя спокойствия» | 7 дней по 2 мин дыхания | +100 XP, бейдж |
| 2 | 🏃 «Шагоход» | 70K шагов за неделю | +100 XP, бейдж |
| 3 | 😴 «Мастер сна» | 7 дней сон >7ч | +100 XP, бейдж |
| 4 | 🧠 «Эмоциональный IQ» | 7 mood check-ins + avg Emotional >75 | +100 XP, бейдж |

### 3.3 Duo Challenges (1v1)

**Механика:**
1. Пользователь нажимает «Вызвать на дуэль» → выбирает друга
2. Друг получает push: «🎯 Алекс вызывает тебя! WVI-дуэль на 7 дней»
3. 7 дней соревнования по среднему WVI
4. Победитель: бейдж «Дуэлянт» + micro-reward (1–5 Wellex Points)
5. Sharing: «Я победил в WVI-дуэли! wellex.ai 🏆»

**Вирусный потенциал:**
- Интеграция с мессенджерами (Telegram, WhatsApp invite links)
- Shareable результат (WVI Story card)
- «Вызываю тебя на дуэль WVI!» — уникальный engagement механизм

### 4.2 Лиги (Duolingo-модель)

```
🏆 Diamond League (Топ 1%)
    ↑
💎 Ruby League (Топ 5%)
    ↑
🥇 Gold League (Топ 15%)
    ↑
🥈 Silver League (Топ 30%)
    ↑
🥉 Bronze League (Топ 50%)
    ↑

```

- Каждую неделю: топ-10 из каждой лиги → повышение. Нижние 10 → понижение
- Награды по лигам: XP bonus, эксклюзивные бейджи
- Diamond League: эксклюзивный Living NFT glow effect

## 7. Модерация и безопасность

### 7.1 AI Moderation

| Уровень | Механизм | Действие |
|---------|----------|----------|
| **Auto-filter** | NLP токсичность, спам, фишинг | Блокировка + уведомление |
| **Reporting** | Пользователь жалуется на контент | Review queue (< 4ч) |
| **Human review** | Сложные кейсы | Community Manager (< 24ч) |
| **Escalation** | Угрозы, illegal content | Блокировка + правоохранители |

### 7.2 Community Guidelines

1. **Уважение** — никакой токсичности, буллинга, дискриминации
2. **Приватность** — не делиться чужими данными здоровья
3. **Никакого финансового совета** — в APP нет финансовых дискуссий
4. **Никакого MLM-рекрутинга в чатах** — партнёрские функции только в WEB
5. **Позитивная среда** — поддержка > критика

### 7.3 Privacy Controls

| Настройка | Опции | По умолчанию |
|-----------|-------|:------------:|
| WVI видим для | Все / Друзья / Никто | Друзья |

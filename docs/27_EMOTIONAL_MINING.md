## 2. Определение

### 2.1 Формальное определение

**Emotional Mining** — процесс генерации верифицируемой экономической ценности из непрерывных, криптографически подписанных данных о физическом, эмоциональном и ментальном благополучии человека.

### 2.2 Архитектура процесса

```
EMOTIONAL MINING
├── INPUT: Забота о здоровье
│ ├── Эмоциональное благополучие (HRV, EDA → 11 эмоций)
│ ├── Качество восстановления (сон, recovery)
│ └── Физическая активность (шаги, тренировки)
│
├── PROCESS: Верификация и оценка
│ ├── Wellex Band → TEE-подпись данных
│ ├── ML-классификация → WVI (0–100)
│ ├── Серверный расчёт (anti-cheat)
│ └── On-chain WVI commitment (Oracle)
│
├── OUTPUT: Экономическая ценность
│ ├── Monthly Yield 0–20% (из реального DeFi)
│ ├── NFT Achievements (Soulbound)
│ ├── Статус и репутация в комьюнити
│ └── Data value (агрегированный, с согласия пользователя)
│
└── PROOF: Proof of Wellness
 ├── Криптографическая подпись (TEE)
 ├── Кросс-валидация 11 метрик
 ├── Биометрический fingerprint
 └── Fraud scoring (0–100)
```

## 4. Экономическая модель

### 4.1 Цепочка создания ценности

```
Пользователь заботится о здоровье
 ↓
WVI растёт (0 → 100)
 ↓
Высокий WVI → высокий Monthly Yield (0% → 20%)
 ↓
Monthly Yield из реального DeFi + перераспределение подписок
 ↓
Пользователь получает доход
 ↓
Мотивация к дальнейшей заботе о здоровье
 ↓
Flywheel (самоусиливающийся цикл)
```

### 4.2 Flywheel Effect

```
Больше пользователей
 → Больше подписок ($19/мес)
 → Больше Yield Reserve
 → Привлекательнее Monthly Yield
 → Ещё больше пользователей
 → Больше данных для AI
 → Лучшие рекомендации
 → Выше WVI
 → Выше retention
```

### 4.3 Устойчивость модели

| Фактор | Описание |
|--------|----------|
| **Независимость от притока** | Yield генерируется из DeFi-протоколов + подписок, а не за счёт новых участников |
| **7-ступенчатая шкала WVI** | WVI 0–39 = 0% Monthly Yield; WVI 40–49 = 2%; до 20% при WVI 90–100. Dynamic Yield Cap обеспечивает устойчивость |
| **Реальные DeFi-протоколы** | Aave, Compound, Curve, US Treasuries — проверенная инфраструктура |

## 3. Naming & Vocabulary

### Термины, которые Wellex вводит в обращение

| Термин | Определение | Статус |
|--------|-------------|--------|
| **Emotional Mining** | Процесс генерации дохода из верифицированных данных о благополучии | Trademark pending |
| **WVI (Wellex Vitality Index)** | Индекс здоровья 0–100 из трёх компонентов | Собственная метрика |
| **Yield from Wellness** | DeFi-yield, размер которого зависит от WVI пользователя | Ключевой нарратив |
| **Wellness-to-Earn** | Категория, которую Wellex создаёт — здоровый образ жизни как источник дохода | Отличие от Move-to-Earn |

### Чего избегать в коммуникациях

| ❌ Избегать | ✅ Говорить вместо |
|------------|------------------|
| «Гарантированный доход» | «Target Yield — до 20%/мес при WVI 90+» |
| «Пассивный доход» (без оговорок) | «DeFi-доход, связанный с вашим WVI» |
| «Инвестиции» | «Депозит в стейблкоинах» |
| «Токен Wellex» | «USDC rewards из WellexVault» |
| «MLM» | «Partner Network» / «Партнёрская программа» |
| «Стейкинг» в APP | APP — wellness-only. Стейкинг — только WEB |

## КОНТЕКСТ ПРОДУКТА

Wellex — Web3 Wellness-протокол. Пользователь носит браслет Wellex Band, который отслеживает биометрику и рассчитывает **WVI (Wellex Vitality Index, 0–100)**. Чем выше WVI — тем выше доходность депозита (0–20%/мес). Платформа имеет **MLM-партнёрскую программу** (10 рангов, 10 уровней глубины, 8 потоков дохода).

**Принцип:** APP (iOS/Android) = wellness + community. WEB (этот проект) = финансы + партнёрство + контент.

### МОДУЛЬ 2: Кошелёк (Wallet)
- Мультисетевой баланс (ETH / Polygon / BSC / Arbitrum / Optimism / Base / Avalanche)
- **Пополнение (Crypto):** USDC/USDT → адрес по сети + QR + real-time статус
  - Поля: `deposit_address`, `tx_hash`, `amount_usdc`, `network`, `confirmations`, `status`
- **Пополнение (Card):** MoonPay / Transak SDK, Visa/MC/ApplePay/GooglePay, конвертация USD→USDC

- **Вывод:** выбор суммы + адрес + сеть + preview (net amount - gas) + 2FA + Manual Review если >$5K
  - Статусы: pending_approval / approved / broadcasting / confirmed / failed
  - Поля: `withdrawal_address`, `amount_usdc`, `network`, `gas_fee`, `net_amount`, `tx_hash`
- **История транзакций:** таблица с полями:
  - `date`, `type` (DEPOSIT/WITHDRAWAL/YIELD/REINVEST/PARTNER_BONUS/SUBSCRIPTION_FEE), `amount_usdc`, `network`, `tx_hash`, `fee_usdc`, `wvi_at_time`, `yield_rate`, `status`, `method`
  - Фильтры: тип / дата / сеть / статус / сумма

### МОДУЛЬ 3: Стейкинг (WellexVault)
- Карточка WellexVault: депозит / накопленный yield / реинвест / WVI → Monthly Yield % / lock-up дата / страховка / AI Auto-Optimize ✅
- Калькулятор: сумма + lock-up + WVI → прогноз yield / мес и / год
- «Что будет если улучшить WVI»: динамические подсказки
- История начислений: дата / WVI среднее / yield% / сумма / действие (вывод/compound)
- Управление реинвестом ($129 auto-compound)
- Статус подписки $19/мес: следующее списание / метод / история платежей

### МОДУЛЬ 4: Партнёрская сеть (MLM Network)

  - Popup на клик: личный стейкинг / оборот структуры / активных партнёров / до следующего ранга
  - Lazy load: 3 уровня сразу, 4–10 по запросу, WebSocket real-time обновления

- Аналитика воронки: Приглашений → Переходов → Регистраций → Активных → Стейкинг

**AI-стратег (персонализированный план роста):**

- Персональные скрипты (по данным конкретного партнёра):

  - Мотивация перед апгрейдом партнёра («тебе до Elite $1,200 PV — вот план»)

  - Симулятор возражений (15 возражений × 3 варианта + AI-оценка твоего ответа)

## КЛЮЧЕВЫЕ ТРЕБОВАНИЯ К РЕАЛИЗАЦИИ

1. **Real-time:** WebSocket для D3 дерева, транзакций, admin dashboard метрик

5. **AI everywhere:** AI алерты по сети, AI ассистент, AI контент-генератор, AI fraud detection — всё через GPT-4o API
6. **Audit trail:** все Admin-действия логируются, неизменяемо
7. **Dual approval:** выводы > $10K и bulk-операции > $100K требуют двух Adminов
8. **Mobile-responsive:** Tailwind breakpoints: mobile (<768px) / tablet / desktop / wide (>1440px)
9. **i18n:** RU / EN / ES / PT / ID / AR
10. **Security:** 2FA обязательно для Admin panel, rate limiting, CSRF protection, CSP headers
